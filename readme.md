# Starter Vite Vanilla JS

For fast init static projects or Test Dev :

- Support HMR, SCSS, GSAP, and PWA.
- Work Well with Node V17.4.0.


## How to Use

Install :

```
npm i
```

Use for Dev :

```
npm run dev
```

Use for prod :

```
npm run build
```
